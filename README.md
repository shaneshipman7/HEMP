# 🚨 Texas THC Response Builder

A simple tool to help Texans quickly contact their state representatives about recent THC regulation changes.

## ✅ Use It Here:
👉 https://shaneshipman7.github.io/HEMP/

## ⏱ Takes Less Than 60 Seconds

- Enter your info  
- Customize your message  
- Tap “Open Email”  
- Send  

## 💬 Why This Matters

Recent regulatory changes may impact:
- Small businesses  
- Jobs  
- Consumer access  

Public input helps lawmakers understand real-world impact.
